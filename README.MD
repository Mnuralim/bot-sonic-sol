## Langkah-langkah

### 1. Clone Repositori

```bash
git clone https://github.com/username/bot-pengiriman-sol.git
cd bot-pengiriman-sol
```

### 2. Instal Dependensi

```bash
npm install
```

### 3. Konfigurasi .env File

buat file .env di root projek

```bash
SEED_PHRASE="your seed phrase here"
```

### 4. Run script

```bash
npm run start
```

\*pastikan udah install node js
